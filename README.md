# Excel-Data-Cleaning-Project

Project Overview - 
This project demonstrates end-to-end data cleaning and preprocessing using Microsoft Excel, a core skill required for Data Analyst roles.
The dataset contains missing values, duplicates, inconsistent formats, and text issues, simulating real-world raw data scenarios.
The objective is to transform raw, unstructured data into clean, consistent, and analysis-ready data using Excel techniques.

Objectives -
Identify and handle missing values
Remove duplicate records
Standardize text and date formats
Validate numerical data
Document data quality issues
Create a clean dataset suitable for analysis

Data Cleaning Steps Performed - 
Created a backup of raw data
Applied filters and froze header rows
Identified missing values using conditional formatting
Handled missing data using logical replacements
Removed duplicate records using key identifiers
Cleaned text fields using TRIM() and PROPER()
Standardized date formats
Validated numeric columns (Quantity, Price)
Created a separate Cleaned_Data sheet
Added a Data_Quality_Notes column to document issues
